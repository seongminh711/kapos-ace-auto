정비를 “설명”까지 하는 정비소

고객님의 이해가 곧 신뢰라고 생각합니다. 증상에 맞춰 부품과 작업 내용을 최대한 쉽게 설명드리겠습니다.

예약 문의 증상 선택

오늘의 빠른 안내

  • 경고등 점등 (계기판/공기압)
  • 시동 지연 / 시동불량
  • 떨림 / 출력 저하
  • 브레이크 밀림 / 소음
  • 에어컨 냄새 / 풍량 약함
  • 핸들 떨림 / 쏠림
  • 타이어 편마모
  • 누유(바닥 오일자국)
  • 소음(하체/엔진/배기)

※ 실제 진단은 차량 상태에 따라 달라질 수 있어요.

증상으로 시작

아래에서 증상을 고르면 관련 부품/설명으로 이어집니다.

증상 선택 결과 안내

위에서 증상을 클릭하면, 점검 포인트·가능 원인·권장 조치가 여기에 표시됩니다.

정비 진행 방식

1) 접수증상 확인
2) 진단스캔/점검
3) 견적항목/비용 안내
4) 작업교환/수리
5) 검수최종 확인

※ 고객이 이해할 수 있도록 “설명”을 먼저 드립니다.

부품 설명

점화플러그

점화 계통 핵심 부품. 교체 주기와 상태를 사진/3D 이미지로 설명하면 이해가 빨라집니다.

여기에 Blender 캡처 이미지나 설명 링크를 넣으면 좋아요.

에어컨 필터

냄새/풍량/알레르기와 직결. 필터 오염도 사진을 보여주면 설득력이 큽니다.

나중에 “차종별 호환”도 표로 확장 가능.

브레이크 패드

제동력과 직결되는 핵심 소모품입니다. 마모가 진행되면 소음과 제동거리 증가가 생길 수 있습니다.

브레이크 소음·밀림 증상과 연결해서 설명하면 좋습니다.

브레이크 디스크

패드와 맞물려 차량을 멈추게 하는 부품입니다. 편마모 시 떨림, 소음, 핸들 진동이 발생할 수 있습니다.

브레이크 밀림·진동 증상과 함께 보여주기 좋습니다.

배터리

시동과 전장 시스템의 기본 전원을 담당합니다. 전압이 약해지면 시동 지연이나 각종 경고등이 나타날 수 있습니다.

시동 지연·경고등 증상과 연결하기 좋습니다.

스타터 모터

시동을 걸 때 엔진을 처음 돌려주는 장치입니다. 딸깍 소리만 나고 시동이 안 걸릴 때 주요 점검 대상입니다.

3D 설명 섹션과 연결하면 가장 설득력이 좋습니다.

타이어

접지력과 승차감에 직접 영향을 주는 부품입니다. 편마모, 공기압 이상, 진동, 쏠림 증상과 관련이 깊습니다.

핸들 떨림·쏠림·편마모 증상과 연결하기 좋습니다.

서스펜션 / 하체 부싱

승차감과 하체 안정성을 담당합니다. 노후되면 덜컹거림, 삐걱거림, 하체 소음이 발생할 수 있습니다.

소음 증상과 연결해서 설명하면 신뢰도가 올라갑니다.
